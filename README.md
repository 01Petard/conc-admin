`服务端`：https://github.com/01Petard/conc-server

<hr>
## 管理端技术栈：

- Nodejs版本：22.14.0
- 软件包管理器：pnpm

- Vue2
- Webpack
- ElementUI（不是ElementUI Plus）
- Vue Router

## 项目启动与部署

```shell
pnpm install
```

```shell
pnpm run dev
```

```shell
pnpm run build
```
